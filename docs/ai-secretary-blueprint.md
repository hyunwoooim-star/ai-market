## 🔐 기술: OAuth 연동 방식

### 사장님이 하는 것 (간단)
1. **버튼 클릭** ("카카오 연결하기")
2. **로그인**
3. **"허용" 클릭**

→ 끝! 3단계면 연동 완료

### 우리가 미리 하는 것
- 카카오 비즈니스 앱 등록 + 승인
- 네이버 API 파트너 등록
- Meta (인스타) 앱 등록
- Twilio 전화번호 준비

### 헛소리 방지 (Hallucination Prevention)
```
고객 질문
    ↓
지식 베이스에서 검색
    ↓
┌─────────────┬──────────────┐
│ 정보 있음    │ 정보 없음     │
│ → AI 대답   │ → "확인 후    │
│             │    답변드릴게요" │
│             │ → 사장님 알림  │
└─────────────┴──────────────┘
```
