---

### 2. 카카오톡 채널 ⚠️ 복잡

#### 가능한 것:
- ✅ 채널 추가 요청 (followChannel API)
- ✅ 채널 친구 관계 조회
- ✅ 고객파일 관리 (고객 그룹)
- ✅ 웹훅 알림 (친구 추가/차단)

#### 주의 사항:
| 기능 | 방식 | 비용 |
|-----|-----|-----|
| 채널 친구 추가 | OAuth (개발자 API) | 무료 |
| **메시지 발송** | 비즈메시지 (알림톡/친구톡) | **유료 (건당)** |

**⚠️ 중요 발견:**
- "카카오톡 채널 API"와 "카카오 비즈메시지"는 **다른 서비스**
- OAuth로 채널 연결은 가능하지만, **대량 메시지 발송**은 비즈메시지 필요
- 비즈메시지 비용: 알림톡 ~15원/건, 친구톡 ~20원/건

**필요 조건:**
1. 비즈 앱 전환 (사업자등록증)
2. 비즈니스 채널 전환 (사업자등록증)
3. 앱-채널 연결 신청 (동일 사업자)
4. 승인 기간: **3-5 영업일**

**결론:** 채널 연동은 OAuth 가능, **메시지 발송은 별도 비용 발생**

### 3. 인스타그램 ✅ 가능 (조건부)

| 항목 | 내용 |
|-----|-----|
| **방식** | Facebook/Meta OAuth |
| **필요 조건** | Instagram Business/Creator 계정 + Facebook 페이지 연결 |
| **가능 기능** | 피드 포스팅, 스토리, 댓글 관리 |
| **난이도** | 🟡 중간 |

**필요 권한:**
- `instagram_content_publish` - 포스팅
- `instagram_basic` - 기본 정보
- `instagram_manage_comments` - 댓글 관리

**주의 사항:**
- 개인 계정 ❌ → Business/Creator 계정만 지원
- Facebook 페이지 연결 필수
- Meta 앱 검토 필요 (복잡할 수 있음)

**결론:** Business 계정 한정으로 **원클릭 연동 가능**

## 🚨 핵심 발견: 카카오 메시지 비용

**예상과 다른 부분:**
- OAuth로 "채널 연결"은 가능
- 하지만 **"채널 친구에게 메시지 발송"**은 비즈메시지 서비스 필요
- **비용 발생!**

**비즈메시지 가격 (2024 기준):**
| 유형 | 비용 |
|-----|-----|
| 알림톡 | ~15원/건 |
| 친구톡 텍스트 | ~20원/건 |
| 친구톡 이미지 | ~30원/건 |

**대응 방안:**
1. **사용량 과금** - 고객에게 메시지 비용 전가
2. **플랜에 포함** - 월 X건 무료, 초과분 과금
3. **카카오 채널 파트너센터** - 직접 발송은 무료 (수동)
